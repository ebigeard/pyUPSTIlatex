# Configuration de pyUPSTIlatex

Ce dossier contient les fichiers de configuration par défaut de pyUPSTIlatex.

- `config.default.toml` ne doit pas être modifié à la main, mais est commenté pour expliquer chaque paramètre
- `pyUPSTIlatex.json` peut-être mis à jour avec :

```bash
pyupstilatex update-config
```

Vous pouvez en savoir plus en consultant le wiki : [Configuration et personnalisation](https://github.com/ebigeard/pyUPSTIlatex/wiki/Configuration-et-personnalisation)
